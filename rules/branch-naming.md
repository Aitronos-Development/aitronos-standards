# Branch Naming Rules

To maintain an organised workflow, follow a standard naming convention for branches:
- **Feature**: `feat/new-user-interface`
- **Bug Fix**: `fix/login-issue`
- **Hotfix**: `hotfix/critical-bug`
- **Refactoring**: `refactor/cleanup-routing-logic`
- **Documentation**: `docs/update-readme`
- **Platform Specific**: `feat/ios/login-screen`
- **Issue Tracking**: `feat/[120]-new-login-screen`
